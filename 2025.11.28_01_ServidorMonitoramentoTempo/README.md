Situação de Aprendizagem 1: Configuração do Ambiente Back-End e Fundamentos de Roteamento
Objetivo: Instalação e configuração do ambiente Node.js, Express e ferramentas de desenvolvimento. Introdução ao roteamento (rotas e métodos HTTP) para Back-End.

Servidor de Monitoramento de Tempo

Os alunos devem configurar um servidor Node.js utilizando Express que exponha uma rota simples (ex: /status). Esta rota deve retornar um JSON com a hora atual do servidor e uma mensagem de "Serviço Operacional". O objetivo é praticar a configuração do ambiente, a inicialização do servidor (Node.js/Express) e a criação de rotas básicas de Back-End.
